## How to Take Screenshots:

### For Best Results:
1. Use a consistent browser window size (1920x1080 recommended)
2. Clear browser zoom (100%)
3. Use demo data (run `npm run seed-demo` first)
4. Capture with active/interesting data showing
5. Use browser DevTools device emulation for mobile views

### Tools:
- **Windows:** Win + Shift + S (Snipping Tool)
- **Mac:** Cmd + Shift + 4
- **Chrome Extension:** Full Page Screen Capture
- **Professional:** Lightshot, ShareX

### Naming Convention:
- Use lowercase
- Use hyphens for spaces
- Be descriptive: `loan-payment-receipt.png`

### Image Optimization:
- Format: PNG for UI, JPG for photos
- Optimize size: Use TinyPNG or Squoosh
- Max width: 1920px
- Keep file size under 500KB per image
